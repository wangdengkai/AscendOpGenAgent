# 简介

**页面ID:** atlasopapi_07_00052  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/API/basicdataapi/atlasopapi_07_00052.html

---

本类是一个POD类，在一块连续的内存空间中存储描述信息以及实际的数据元素（元素类型由模板参数T指定）。不支持动态扩容。

本类的描述信息包括：最大可保存的元素个数、当前存放的实际元素数量等。

