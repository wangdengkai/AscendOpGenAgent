# 进程中断问题现象描述

**页面ID:** troubleshooting_0061  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/maintenref/troubleshooting/troubleshooting_0061.html

---

应用进程异常退出，一般主要由于以下原因导致：

- 接口报错：plog日志中是否存在"acl***" 、“[drv api]”、“rt***”等接口报错；
- 任务执行出错：plog日志中存在“fault kernel_name”、“Task run failed”等报错；
- TaskScheduler CPU心跳丢失：plog日志中存在“Device lost heartbeat”报错，且黑匣子日志存在“HEARTBEAT EXCEPTION”关键报错；
- Control CPU心跳丢失：syslog日志中存在“fatal panic”关键报错；
- 内存资源未及时释放：plog日志中存在“RESOURCE_ALLOC_FAIL”报错。
