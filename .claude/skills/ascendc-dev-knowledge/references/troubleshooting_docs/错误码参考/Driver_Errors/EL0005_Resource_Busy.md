# EL0005 Resource_Busy

**页面ID:** atlaserrorcode_15_0268  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/maintenref/troubleshooting/atlaserrorcode_15_0268.html

---

#### Symptom

The resources are busy.

#### Possible Cause

1. The resources have been occupied.
2. The device is being reset.
3. Software is not ready.

#### Solution

1. Close applications not in use.
2. Wait for a while and try again.
