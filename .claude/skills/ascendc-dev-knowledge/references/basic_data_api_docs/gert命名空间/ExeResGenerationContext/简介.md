# 简介

**页面ID:** atlasopapi_07_00690  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/API/basicdataapi/atlasopapi_07_00690.html

---

ExeResGenerationContext继承自ExtendedKernelContext，用于生成执行资源的上下文类，主要用于在算子执行前，生成或配置执行所需的各种资源，包括：内存、异步执行流、同步资源、算子属性、输入输出张量的形状信息。

```text
               ↑

```
