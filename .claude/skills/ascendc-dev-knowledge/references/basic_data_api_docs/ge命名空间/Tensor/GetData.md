# GetData

**页面ID:** atlasopapi_07_00462

#### 函数功能

获取Tensor中的数据。

const uint8_t* GetData() const返回的数据不可修改，uint8_t* GetData()返回的数据可修改。

#### 函数原型

```
const uint8_t *GetData() const
uint8_t *GetData()
```

#### 参数说明

无。

#### 返回值

Tensor中所存放的数据。

#### 异常处理

无。

#### 约束说明

无。
