# 简介

**页面ID:** atlasopapi_07_00556

TilingData用于存储算子Tiling的计算结果，即Tiling实现根据算子的shape等信息确定的数据切分算法相关参数（比如每次搬运的数据块大小，搬运、计算操作的循环次数）。

#### 需要包含的头文件

```
#include <tiling_data.h>
```
