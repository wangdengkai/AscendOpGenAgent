# 简介<a name="ZH-CN_TOPIC_0000002554343561"></a>

本章节API功能为对输入数据做类型转换，并遵循规定的精度转换处理模式，返回转换后的值。浮点数的表示方式和舍入规则请见[功能说明](Cast.md#section618mcpsimp)。

类型转换结果受舍入模式和饱和模式影响。

## 控制饱和行为的方式<a name="section1194916101549"></a>

饱和行为受寄存器CTRL\[60\]、CTRL\[48\]和api同时控制，CTRL寄存器的设置请参考[SetCtrlSpr\(ISASI\)](SetCtrlSpr(ISASI).md)接口。CTRL\[60\]和CTRL\[48\]

宽范围数据转窄范围数据且转换目标类型不为float时，包括即为\(float转half\)、\(float转bfloat16\_t\)、\(bfloat16\_t转half\)或\(float、half转hifloat8\_t、fp8\_e4m3fn\_t、fp8\_e5m2\_t\)，接口的饱和行为受硬件寄存器CTRL\[60\]和CTRL\[48\]的影响。具体行为如下：

-   当CTRL\[60\] = 0时，饱和行为受接口独立控制。
    -   后缀为\_\_sat的接口转换结果为饱和模式。
    -   不带\_\_sat后缀的接口转换结果为非饱和模式。

-   当CTRL\[60\] = 1时，饱和行为只受CTRL\[48\]全局控制。
    -   当CTRL\[48\] = 1，接口转换结果为非饱和模式。
    -   当CTRL\[48\] = 0，接口转换结果为饱和模式。
