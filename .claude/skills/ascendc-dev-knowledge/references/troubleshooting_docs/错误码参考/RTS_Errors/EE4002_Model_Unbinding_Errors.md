# EE4002 Model_Unbinding_Errors

**页面ID:** atlaserrorcode_15_0183  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/maintenref/troubleshooting/atlaserrorcode_15_0183.html

---

#### Symptom

Failed to unbind the stream to the model. %s

#### Possible Cause

1. The stream to be unbound is not bound to the model.
2. The model is running.

#### Solution

1. Check the code to ensure that the stream to be unbound is bound to the model.
2. Ensure that the model is not running.
