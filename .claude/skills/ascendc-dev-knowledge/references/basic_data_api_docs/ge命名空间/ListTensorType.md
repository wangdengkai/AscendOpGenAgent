#### 函数功能

ListTensorType类用以定义输入或者输出支持的数据类型，是TensorType的封装，用于标识支持多个数据类型的情况。

#### 函数原型

```
explicit ListTensorType(const TensorType &type)
```

#### 参数说明

| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| type | 输入 | 数据类型，TensorType。 |

#### 返回值

无。

#### 异常处理

无。
