жҢүе…ғзҙ еҒҡеҸҚйҮҸеҢ–и®Ўз®—пјҢжҜ”еҰӮе°Ҷint32\_tж•°жҚ®зұ»еһӢеҸҚйҮҸеҢ–дёәhalf/floatзӯүж•°жҚ®зұ»еһӢгҖӮ**жң¬жҺҘеҸЈжңҖеӨҡж”ҜжҢҒиҫ“е…ҘдёәдәҢз»ҙж•°жҚ®пјҢдёҚж”ҜжҢҒжӣҙй«ҳз»ҙеәҰзҡ„иҫ“е…ҘгҖӮ**

-   еҸҚйҮҸеҢ–зі»ж•°deqScaleеҸҜд»Ҙдёәж ҮйҮҸжҲ–иҖ…еҗ‘йҮҸпјҢдёәеҗ‘йҮҸзҡ„жғ…еҶөдёӢпјҢcalCount <= deqScaleзҡ„е…ғзҙ дёӘж•°пјҢеҸӘжңүеүҚCalCountдёӘеҸҚйҮҸеҢ–зі»ж•°з”ҹж•Ҳпјӣ
-   иҫ“еҮәdstTensorзҡ„shapeдёә**пјҲm, n\_dstпјү**пјҢ n \* sizeof\(dstT\)дёҚж»Ўи¶і32еӯ—иҠӮеҜ№йҪҗж—¶пјҢйңҖиҰҒ**еҗ‘дёҠиЎҘйҪҗдёә32еӯ—иҠӮ**пјҢn\_dstдёәеҗ‘дёҠиЎҘйҪҗеҗҺзҡ„еҲ—ж•°гҖӮ

дёӢйқўйҖҡиҝҮдёӨдёӘе…·дҪ“зҡ„зӨәдҫӢжқҘи§ЈйҮҠеҸӮж•°зҡ„й…ҚзҪ®е’Ңи®Ўз®—йҖ»иҫ‘пјҲдёӢж–ҮдёӯDequantParamsзұ»еһӢдёәеӯҳеӮЁshapeдҝЎжҒҜзҡ„з»“жһ„дҪ“\{m, n, calCount\}пјүпјҡ
