# 构造&拷贝构造函数<a name="ZH-CN_TOPIC_0000002523303590"></a>

## 功能说明<a name="zh-cn_topic_0000001600307125_zh-cn_topic_0000001526594958_zh-cn_topic_0000001525424352_section36583473819"></a>

构造算子参数的OpParamDef，OpParamDef用于描述算子参数的信息，如数据类型、格式等。一般用户不会直接构造OpParamDef，而是通过Input、Output接口间接构造算子参数的OpParamDef。

## 函数原型<a name="zh-cn_topic_0000001600307125_zh-cn_topic_0000001526594958_zh-cn_topic_0000001525424352_section13230182415108"></a>

```
explicit OpParamDef(const char *name);
OpParamDef(const OpParamDef &def);
```
