# E21001 File_Operation_Error_Open

**页面ID:** atlaserrorcode_15_0123  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/maintenref/troubleshooting/atlaserrorcode_15_0123.html

---

#### Symptom

Failed to open file %s. Reason: %s.

#### Possible Cause

1. The file does not exist.
2. Insufficient file permissions.
3. Package installation error.

#### Solution

1. Configure the file path correctly.
2. Configure the file permissions correctly.
3. Reinstall the package.
