# EL0007 No_Permission

**页面ID:** atlaserrorcode_15_0270  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/maintenref/troubleshooting/atlaserrorcode_15_0270.html

---

#### Symptom

You have no permission.

#### Possible Cause

1. The resources have been occupied.
2. You have no permission on this API.

#### Solution

1. Close applications not in use.
2. Check that you have the required permission.
