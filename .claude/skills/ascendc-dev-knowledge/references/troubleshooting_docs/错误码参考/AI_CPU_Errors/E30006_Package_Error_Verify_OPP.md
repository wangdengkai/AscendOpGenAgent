# E30006 Package_Error_Verify_OPP

**页面ID:** atlaserrorcode_15_0130  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/maintenref/troubleshooting/atlaserrorcode_15_0130.html

---

#### Symptom

Failed to verify the OPP.

#### Possible Cause

1. The AI CPU OPP on the host is tampered with.
2. An ECC memory error occurs on the device.

#### Solution

1. Install the correct OPP.
2. Obtain the device log, check whether ECC errors are reported, and contact technical support at https://www.hiascend.com/support.
