# EB1000 Compilation_Error_Invalid_Primitives

**页面ID:** atlaserrorcode_15_0156  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/maintenref/troubleshooting/atlaserrorcode_15_0156.html

---

#### Symptom

Primitives check failed. Reason: %s

#### Possible Cause

1. The Primitives are not properly used or not aligned.
2. Incorrect parameter types are used.
3. The correct primitive is not punched on the corresponding axis.

#### Solution

Check the schedule and compute statements to ensure that the schedule is correct.
