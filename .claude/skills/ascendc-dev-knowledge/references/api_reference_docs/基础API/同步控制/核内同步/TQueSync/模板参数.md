#### 功能说明

TQueSync类的模板参数用于指定源流水和目标流水，目的流水等待源流水。

#### 函数原型

```
template<pipe_t src, pipe_t dst>
class TQueSync {
public:
    __aicore__ inline void SetFlag(TEventID id);
    __aicore__ inline void WaitFlag(TEventID id);
};
```

#### 参数说明

**表1 **参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| src | 输入 | 源流水。 支持的流水参考表1。 |
| dst | 输入 | 目的流水。 支持的流水参考表1。 |
