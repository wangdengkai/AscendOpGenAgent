# OpReceiver构造函数和析构函数

**页面ID:** atlasopapi_07_00382  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/API/basicdataapi/atlasopapi_07_00382.html

#### 函数功能

OpReceiver构造函数，接收自定义算子的注册信息。

#### 函数原型

```
OpReceiver(OpRegistrationData &reg_data)
~OpReceiver()
```

#### 参数说明

| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| reg_data | 输入 | 需要注册的算子信息。 |

#### 返回值

OpReceiver构造函数返回OpReceiver类型的对象。

#### 异常处理

无。

#### 约束说明

无。
