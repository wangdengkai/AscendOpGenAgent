# GetShapeDimNum

**页面ID:** atlasopapi_07_00471  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/API/basicdataapi/atlasopapi_07_00471.html

#### 函数功能

获取Shape的维度大小，即rank大小。

#### 函数原型

```
size_t GetShapeDimNum() const
```

#### 参数说明

无。

#### 返回值

返回Shape的维度大小，即Shape的rank，如果是unknown的rank，返回0。

#### 异常处理

无。

#### 约束说明

无。
