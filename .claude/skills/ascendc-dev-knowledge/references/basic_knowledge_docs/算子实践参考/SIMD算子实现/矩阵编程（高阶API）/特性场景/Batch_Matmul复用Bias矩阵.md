#### дҪҝз”ЁеңәжҷҜ

Batch MatmulдёӯжҜҸдёӘBatchзҡ„Matmulи®Ўз®—еҸҜд»ҘдҪҝз”ЁзӣёеҗҢзҡ„Biasзҹ©йҳөгҖӮ

#### зәҰжқҹиҜҙжҳҺ

AгҖҒBгҖҒCзҹ©йҳөзҡ„Layoutзұ»еһӢйғҪдёәNORMALж—¶пјҢдёҚж”ҜжҢҒbatchModeеҸӮж•°и®ҫдёәSINGLE_LARGE_THAN_L1пјҢеҚіBiasеӨҚз”ЁеңәжҷҜдёӢпјҢеҚ•Batchзҡ„AгҖҒBзҹ©йҳөж•°жҚ®жҖ»е’ҢдёҚеҫ—и¶…иҝҮL1 Bufferзҡ„еӨ§е°ҸгҖӮ

#### и°ғз”ЁзӨәдҫӢ

е®Ңж•ҙзҡ„з®—еӯҗж ·дҫӢиҜ·еҸӮиҖғ[BatchMatmulеӨҚз”ЁBiasз®—еӯҗж ·дҫӢ](https://gitee.com/ascend/ascendc-api-adv/tree/master/examples/matrix/batch_matmul_bias_no_batch)гҖӮ

```
// иҮӘе®ҡд№үMatmulConfigеҸӮж•°пјҢе°Ҷе…¶дёӯзҡ„isBiasBatchеҸӮж•°и®ҫзҪ®дёәfalseпјҢдҪҝиғҪBatchMatmulзҡ„BiasеӨҚз”ЁеҠҹиғҪгҖӮ
constexpr MatmulConfigMode configMode = MatmulConfigMode::CONFIG_NORM;
constexpr MatmulBatchParams batchParams = {
  false, BatchMode::BATCH_LESS_THAN_L1, false /* isBiasBatch */
};
constexpr MatmulConfig CFG_MM = GetMMConfig<configMode>(batchParams);
AscendC::Matmul<A_TYPE, B_TYPE, C_TYPE, BIAS_TYPE, CFG_MM> mm;

REGIST_MATMUL_OBJ(&pipe, GetSysWorkSpacePtr(), mm, &tiling); // еҲқе§ӢеҢ–matmulеҜ№иұЎ
mm.SetTensorA(gm_a);    // и®ҫзҪ®е·Ұзҹ©йҳөA
mm.SetTensorB(gm_b);    // и®ҫзҪ®еҸізҹ©йҳөB
mm.SetBias(gm_bias);    // и®ҫзҪ®BiasпјҢзҹ©йҳөеӨ§е°Ҹдёә1 * singleCoreN
mm.IterateBatch(gm_c, batchA, batchB, false);
mm.End();
```
