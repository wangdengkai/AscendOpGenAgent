# GetOriginFormat

**页面ID:** atlasopapi_07_00466  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/API/basicdataapi/atlasopapi_07_00466.html

---

#### 函数功能

获取Tensor的原始Format。

该Format是指原始网络模型的Format。

#### 函数原型

```
ge::Format GetOriginFormat() const
```

#### 参数说明

无。

#### 返回值

返回Tensor的原始Format值，默认值为FORMAT_RESERVED。

#### 异常处理

无。

#### 约束说明

无。
