#### 函数功能

向TensorDesc中设置Tensor的Format。

#### 函数原型

```
void SetFormat(Format format)
```

#### 参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| format | 输入 | 需设置的Format信息。 关于Format类型，请参见Format。 |

#### 返回值

无。

#### 异常处理

无。

#### 约束说明

无。
