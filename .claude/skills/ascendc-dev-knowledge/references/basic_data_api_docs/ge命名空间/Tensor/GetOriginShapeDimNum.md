# GetOriginShapeDimNum

**页面ID:** atlasopapi_07_00468  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/API/basicdataapi/atlasopapi_07_00468.html

---

#### 函数功能

获取原始Shape的维度大小，即rank大小。

#### 函数原型

```
size_t GetOriginShapeDimNum() const
```

#### 参数说明

无。

#### 返回值

返回原始Shape的维度大小，即原始Shape的rank，如果是unknown的rank，返回0。

#### 异常处理

无。

#### 约束说明

无。
