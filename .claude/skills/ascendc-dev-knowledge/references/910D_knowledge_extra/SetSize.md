## 功能说明<a name="section618mcpsimp"></a>

设置当前LocalTensor Size大小。单位为元素。当用户重用local tensor变量且使用长度发生变化的时候，需要使用此接口重新设置Size。

## 函数原型<a name="section620mcpsimp"></a>

```
__aicore__ inline void SetSize(const uint32_t size)
```
