# GetDim

**页面ID:** atlasopapi_07_00423  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/API/basicdataapi/atlasopapi_07_00423.html

#### 函数功能

获取Shape第idx维的长度。

#### 函数原型

```
int64_t GetDim(size_t idx) const
```

#### 参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| idx | 输入 | 维度索引，索引从0开始。 |

#### 返回值

第idx维的长度。

#### 异常处理

无。

#### 约束说明

无。
