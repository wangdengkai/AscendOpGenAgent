#### 宏功能

按指定名称注册算子。

#### 宏原型

```
REGISTER_CUSTOM_OP(name)
```

#### 参数说明

| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| name | 输入 | 算子类型名称。 |
