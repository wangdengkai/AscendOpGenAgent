# operator=

**页面ID:** atlasascendc_api_07_1003  
**来源:** https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/850/API/ascendcopapi/atlasascendc_api_07_1003.html

#### 功能说明

赋值运算符重载，实现OpMC2Def类对象的深拷贝。

#### 函数原型

```
OpMC2Def &operator=(const OpMC2Def &def)
```

#### 参数说明

| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| def | 输入 | 被拷贝的OpMC2Def类对象。 |

#### 返回值说明

拷贝后的结果对象的引用。

#### 约束说明

无。
