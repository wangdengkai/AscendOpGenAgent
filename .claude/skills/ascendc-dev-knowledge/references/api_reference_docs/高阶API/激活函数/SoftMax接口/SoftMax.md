```
def softmax(src):
    #еҹәдәҺlastиҪҙиҝӣиЎҢrowmaxпјҲжҢүиЎҢеҸ–жңҖеӨ§еҖјпјүеӨ„зҗҶ
    max = np.max(src, axis=-1, keepdims=True)
    sub = src - max
    exp = np.exp(sub)
    #еҹәдәҺlastиҪҙиҝӣиЎҢrowsumпјҲжҢүиЎҢжұӮе’ҢпјүеӨ„зҗҶ
    sum = np.sum(exp, axis=-1, keepdims=True)
    dst = exp / sum
    return dst, max, sum
```

#### зәҰжқҹиҜҙжҳҺ

- srcе’Ңdstзҡ„Tensorз©әй—ҙеҸҜд»ҘеӨҚз”ЁгҖӮ
- sumTensorе’ҢmaxTensorдёәиҫ“еҮәпјҢе№¶дё”lastиҪҙй•ҝеәҰеҝ…йЎ»еӣәе®ҡ32ByteпјҢйқһlastиҪҙеӨ§е°ҸйңҖиҰҒе’Ңsrcд»ҘеҸҠdstдҝқжҢҒдёҖиҮҙгҖӮ
- sumTensorе’ҢmaxTensorзҡ„ж•°жҚ®зұ»еһӢйңҖиҰҒдҝқжҢҒдёҖиҮҙгҖӮ

- дёҚж”ҜжҢҒsharedTmpBufferдёҺжәҗж“ҚдҪңж•°е’Ңзӣ®зҡ„ж“ҚдҪңж•°ең°еқҖйҮҚеҸ гҖӮ
- еҪ“еҸӮж•°softmaxShapeInfoдёӯsrcM != oriSrcM жҲ–иҖ… srcK != oriSrcKж—¶пјҢејҖеҸ‘иҖ…йңҖиҰҒеҜ№GMдёҠзҡ„еҺҹе§Ӣиҫ“е…Ҙ(oriSrcM, oriSrcK)еңЁMжҲ–Kж–№еҗ‘иЎҘйҪҗж•°жҚ®еҲ°(srcM, srcK)пјҢиЎҘйҪҗзҡ„ж•°жҚ®дјҡеҸӮдёҺйғЁеҲҶиҝҗз®—пјҢеңЁиҫ“е…Ҙиҫ“еҮәеӨҚз”Ёзҡ„еңәжҷҜдёӢпјҢAPIзҡ„и®Ўз®—з»“жһңдјҡиҰҶзӣ–srcTensorдёӯиЎҘйҪҗзҡ„еҺҹе§Ӣж•°жҚ®пјҢеңЁиҫ“е…Ҙиҫ“еҮәдёҚеӨҚз”Ёзҡ„еңәжҷҜдёӢпјҢAPIзҡ„и®Ўз®—з»“жһңдјҡиҰҶзӣ–dstTensorдёӯеҜ№еә”srcTensorиЎҘйҪҗдҪҚзҪ®зҡ„ж•°жҚ®гҖӮ

#### и°ғз”ЁзӨәдҫӢ

жң¬ж ·дҫӢдёӯиҫ“е…Ҙsrcе’Ңиҫ“еҮәdstзҡ„shapeеӨ§е°Ҹдёә[320,64]пјҢдёӯй—ҙи®Ўз®—з»“жһңsumTensorе’ҢmaxTensorзҡ„shapeеӨ§е°Ҹдёә[320,16]пјҢж•°жҚ®зұ»еһӢеқҮдёәhalfпјҢиҫ“е…Ҙиҫ“еҮәзҡ„ж•°жҚ®жҺ’еёғж јејҸдёәNDпјҢsrcе’Ңdstз©әй—ҙдёҚеӨҚз”ЁпјҢдёҚдҪҝиғҪеҹәжң¬еқ—гҖӮжӣҙеӨҡз®—еӯҗж ·дҫӢиҜ·еҸӮиҖғ[softmaxз®—еӯҗж ·дҫӢ](https://gitee.com/ascend/ascendc-api-adv/tree/master/examples/activation/softmax)гҖӮ

```

AscendC::SoftMaxShapeInfo srcShape = {height, width, height, width};
```
